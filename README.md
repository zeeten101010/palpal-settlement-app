# 팔팔너구리해장 월결산 웹앱

식당 손익 확인용 월결산 웹앱입니다.

복식부기, 차변/대변, 전표 시스템이 아니라 **매장 운영 손익 확인**에 맞춘 구조입니다.

## 기능

- 자금일보: 자금현황, 금일 출금완료, 출금예정 직접 입력/수정/삭제
- 출금예정 입력 시 합계, 예상잔액, 미지급 거래처 요약 자동 계산
- 자금일보는 월결산 손익과 별도 저장

- 공용 비밀번호 접속
- 거래 입력
- 자동 저장
- 거래 목록 수정/삭제
- 계정과목 관리
- 월별 손익 집계
- 원가율/인건비율/순이익률 자동 계산
- Excel 다운로드
- PDF 출력
- 비손익거래 분리
- 월결산 분석: 문제점, 개선 방향, 다음달 실행 기준 자동 출력
- 보고용 월간 손익 보고서 출력: 핵심 요약, 매출/비용 현황, 주요 비용률, 전월 비교, 비손익거래 정리

## 기술 스택

- Next.js
- Supabase PostgreSQL
- Vercel
- SheetJS xlsx

## 설치

```bash
npm install
npm run dev
```

## Supabase 세팅

1. Supabase 프로젝트 생성
2. SQL Editor 열기
3. `supabase/schema.sql` 전체 복사 후 실행
4. Project Settings > API에서 아래 값 확인
   - Project URL
   - service_role key

## 환경변수

`.env.example`을 `.env.local`로 복사 후 입력합니다.

```bash
cp .env.example .env.local
```

필수 값:

```env
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
APP_PASSWORD=
SESSION_SECRET=
```

주의:

`SUPABASE_SERVICE_ROLE_KEY`는 절대 `NEXT_PUBLIC_`을 붙이면 안 됩니다.

## 배포

Vercel에 올린 뒤 Environment Variables에 위 환경변수를 그대로 등록하면 됩니다.

## 손익 계산 기준

```text
총매출 = 홀매출 + 배달매출 + 기타매출
손익비용 = 매출원가 + 인건비 + 고정비 + 변동비 + 운영비 + 기타비용
순이익 = 총매출 - 손익비용
비손익거래 = 순이익 계산에서 제외
```

비용 제외 항목:

- 물류보증금 반환
- 가맹보증금 반환
- 부가세 납부
- 대출원금 상환
- 대표자 인출

비용 포함 항목:

- 식자재비
- 인건비
- 임차료
- 배달앱 수수료
- 카드수수료
- 광고비
- 대출 이자
- 소모품비

## v9 자금일보 개선

- 자금현황은 고정 계좌 기준으로 잔액 숫자만 입력
- 금일 출금완료와 출금예정은 엑셀형 셀 입력 후 한 번에 저장
- 자금일보는 월결산 손익과 별도 저장


## v10 모바일 대응

- 자금일보 셀 입력 표가 모바일에서는 카드형 입력 UI로 자동 변환됩니다.
- 메뉴는 모바일에서 가로 스크롤됩니다.
- 요약 카드, 보고서, 분석 화면이 모바일 폭에 맞게 1열로 정리됩니다.


## v11 AI 자금 분석 / PDF 개선

- 자금일보에 AI 자금 진단 카드 추가
- 지급 커버율, 부족 예상액, 최대 지급처 자동 분석
- PDF 출력 시 A4 1페이지 요약 보고서 형태로 출력
- 화면은 미니멀하게 유지하고 출력용 섹션은 별도 구성


## v12 모바일 PDF 저장

- 자금일보 PDF 버튼이 모바일에서도 PDF 파일 저장을 시도합니다.
- jsPDF/html2canvas를 사용해 A4 1페이지 요약 이미지를 PDF로 생성합니다.
- PDF 좌우 여백은 12mm 기준으로 잡았습니다.


## v13 PDF 미리보기 / 색상 수정

- PC와 모바일에서 PDF 미리보기 영역이 화면에 표시됩니다.
- PDF 저장은 미리보기 영역을 그대로 캡처합니다.
- 기존의 글자가 흐리게 나오는 문제를 수정했습니다.
- PDF 저장 여백을 좌우/상하 13mm로 조정했습니다.


## v14 PDF 미리보기 위치/표시 수정

- PDF 미리보기 영역을 자금일보 상단으로 이동했습니다.
- PC/모바일 모두 미리보기가 항상 보이도록 CSS를 강제 수정했습니다.
- 모바일 PDF 저장 후 완료 안내창을 띄웁니다.
- 미리보기 글자 색상과 대비를 다시 보정했습니다.


## v15 월말재고 차감 / 월 손익 분석표

- 지출 > 매출원가 > 월말재고 차감 계정 추가
- 월말재고 금액은 양수로 입력하면 결산 요약에서는 매출원가에서 자동 차감
- 결산 요약 08 월 손익 분석표 추가
- 계정별 매출 대비 %, 상위 계정 대비 %를 표시

기존 DB는 Supabase SQL Editor에서 `supabase/inventory_adjustment_migration.sql`을 1번 실행하세요.


## v16 월말재고 세부 차감

월말재고 차감을 세부 계정으로 나눴습니다.

- 지출 > 매출원가 > 월말재고 차감 > 육류 재고 차감
- 지출 > 매출원가 > 월말재고 차감 > 식자재 재고 차감
- 지출 > 매출원가 > 월말재고 차감 > 음료 재고 차감
- 지출 > 매출원가 > 월말재고 차감 > 주류 재고 차감
- 지출 > 매출원가 > 월말재고 차감 > 기타 재고 차감

입력 금액은 양수로 넣고, 결산 요약에서는 자동으로 매출원가에서 마이너스 차감됩니다.

기존 DB는 Supabase SQL Editor에서 `supabase/inventory_adjustment_detail_migration.sql`을 1번 실행하세요.


## v17 거래처 자동완성

- 거래 입력의 거래처명 입력칸에 이전 거래처 자동완성 추가
- 자금일보 금일 출금완료 / 출금예정 거래처 셀에도 자동완성 추가
- 기존 월결산 거래처와 자금일보 거래처를 합쳐 추천 목록 생성
- DB 변경 없음

- DB 변경 없음
